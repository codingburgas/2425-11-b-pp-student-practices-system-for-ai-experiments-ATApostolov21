About this Project

System for Experiments with Artificial Intelligence

This web-based system allows students and teachers to create, train, and test simple AI models while practicing database work, result visualization, and using basic algorithms.

Project Goals

This platform was created to help students understand and experiment with AI concepts through hands-on learning.

The system enables users to:

  • Access different AI models (perceptron, linear regression, logistic regression, neural network)
  • Upload datasets to train these models
  • Visualize training results (accuracy, graphs, tables)
  • Provide input to trained systems and receive predictions
  • Share and comment on results with other students or teachers

Technical Implementation

The system is built using the following technologies:

  • Flask - Web framework
  • Flask-Login - Session management
  • Flask-SQLAlchemy - Database ORM
  • Flask-Bootstrap - Frontend styling
  • Scikit-learn - Machine learning algorithms
  • Matplotlib/Plotly - Data visualization

User Roles

Student Role

  • Upload datasets
  • Train AI models
  • View and use personal models
  • Make predictions
  • View visualizations

Teacher Role

  • All student capabilities
  • Access to all student-trained models
  • Ability to comment on student work
  • View all datasets and models
